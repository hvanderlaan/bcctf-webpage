# CTF Web

Website for CTF challenges for Brightcubes

## Usage

```bash
git clone https://github.com/hvanderlaan/bcctf-webpage.git
cd ctf-web
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 server.py
http://127.0.0.1:5000
```
